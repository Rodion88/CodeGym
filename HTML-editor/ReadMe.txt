HTML editor in Java

The graphical interface will be a window with a menu and a panel with two tabs.

The first tab will contain a text panel that will render the html page. On it it will be possible to format and edit the text of the page. On the second tab there will be an editor that will display the html code of the page, all the html tags used will be visible in it. It will also be possible to change the text of the page, add and remove various tags.
